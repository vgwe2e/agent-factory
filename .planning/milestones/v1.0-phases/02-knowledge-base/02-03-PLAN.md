The guide contains:
1. Quick Decision Framework - flowchart for LLM requirement -> agent type selection
2. Decision Matrix with three sections:
   - Use Process Builder WHEN: 6 scenarios (deterministic, structured data, transaction control, scheduling, performance-critical, rule-based)
   - Use Agentic AI WHEN: 6 scenarios (NL interface, unstructured data, adaptive reasoning, multi-agent, conversational, autonomous)
   - Use BOTH (Hybrid) WHEN: 5 scenarios (LLM orchestrates procedural, conversational control, process needs reasoning, agent results persist, AI via chat)
3. Scenario-Based Guide - 8 concrete scenarios with recommendations and component lists
4. Component Selection Flowchart - 4-step decision process
5. Integration Patterns - 4 patterns (Agent wraps Process, Process calls Agent Team, Multi-Agent Pipeline, Autonomous Workflow)
6. Validation Checklist

Key mapping for downstream scoring:
- DETERMINISTIC archetype -> typically Process Builder route
- AGENTIC archetype -> typically Agentic AI route
- GENERATIVE archetype -> could be either Agentic AI or Hybrid depending on specifics

The decision criteria that matter for scoring:
- Does the feature require LLM capabilities? (yes/no)
- Is it deterministic/procedural? (-> Process)
- Does it need NL understanding, unstructured data, or adaptive reasoning? (-> Agent)
- Does it combine procedural + reasoning? (-> Hybrid)

4. Create src/types/orchestration.ts:

```typescript
export type OrchestrationRoute = "process" | "agentic_ai" | "hybrid";

export interface DecisionCriterion {
  criterion: string;
  description: string;
  example: string;
}

export interface DecisionScenario {
  name: string;
  recommendation: OrchestrationRoute;
  reason: string;
  components: string[];
}

export interface IntegrationPattern {
  name: string;
  flow: string;
  use_case: string;
}

export interface ArchetypeMapping {
  primary_route: OrchestrationRoute;
  secondary_route: OrchestrationRoute;
  rationale: string;
}

export type LeadArchetype = "DETERMINISTIC" | "AGENTIC" | "GENERATIVE";

export interface DecisionGuide {
  version: string;
  routes: OrchestrationRoute[];
  decision_criteria: Record<OrchestrationRoute, DecisionCriterion[]>;
  scenarios: DecisionScenario[];
  integration_patterns: IntegrationPattern[];
  archetype_mapping: Record<LeadArchetype, ArchetypeMapping>;
}
```
