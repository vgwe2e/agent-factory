     ```
     === Hierarchy ===
     L4 Activities: {count}
     L3 Opportunities: {count}
     Domains:       {unique l1 values from hierarchy, comma-separated}

     === Ready for Processing ===
     ```
  6. Exit with code 0

Format revenue with Intl.NumberFormat('en-US') for readability ($184,992,000,001).
