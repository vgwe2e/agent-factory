    - Retry prompt includes YAML parse or Zod validation error from previous attempt
    - Prompt includes all PB node names, UI component names, and orchestration patterns as reference

       - System prompt: "You are an Aera platform solutions engineer mapping opportunities to Aera components."
       - Include rules: YAML format, 5 sections (streams, cortex, process_builder, agent_teams, ui), drill to specific components + properties, list only (no counts/sizing), include confidence: confirmed|inferred per entry
       - Provide component name glossary: all 22 PB nodes by name, all 21 UI components by name, integration patterns
