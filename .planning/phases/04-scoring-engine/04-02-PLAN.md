  <action>
    Create src/scoring/ollama-client.ts with two exported functions:

    1. ollamaChat(messages, format): wraps Ollama /api/chat endpoint
       - POST to http://localhost:11434/api/chat
       - Body: { model: "qwen2.5:32b", messages, stream: false, format, options: { temperature: 0 } }
       - Timeout: AbortSignal.timeout(120_000) -- 2 minutes per call (32B model is slow)
       - Returns Result type: { success: true, content: string, durationMs: number } | { success: false, error: string }
       - Duration from response.total_duration (nanoseconds) converted to ms
       - Error handling: check response.ok, catch network errors, return error strings (never throw -- per project convention)

    2. scoreWithRetry<T>(schema: ZodSchema<T>, callFn: () => Promise<string>, maxRetries = 3): generic retry wrapper
       - Call callFn(), JSON.parse the result, validate with schema.parse()
       - On failure: wait 1s * 2^attempt, retry
       - Returns: { success: true, data: T } | { success: false, error: string }
       - Log attempt number on retry for debugging

    Constants at top: OLLAMA_CHAT_API, SCORING_MODEL = "qwen2.5:32b", SCORING_TIMEOUT_MS = 120_000, SCORING_TEMPERATURE = 0

    Follow project patterns: Result type (success/error union), no exceptions, error strings include context.
  </action>

  </verify>
  <done>Ollama client compiles, exports ollamaChat and scoreWithRetry. Uses native fetch with format parameter. Returns Result types, never throws.</done>
